[ General Changes ]

* Added a new Profile page that displays a breakdown of recent matches and all-time player and [[Hero|hero]] statistics (Note: the newly added [[Stats|stats]] '[[Souls]] Collected' and '[[Healing]]' are starting at 0 for all players)

* Added Report button to ESC menu player list to allow reporting of teammates in-game
* Improved visibility of report button on post-game screen

* You can now drag-and-drop [[Items|items]] between categories in Builds
* Added search to the Public Builds list, you can search by keyword within build names
* Newly published [[Hero|hero]] builds now store the language of the build
* Added 'Show All Languages' checkbox in Public Builds list, can be unchecked to only see builds in your language
* Fixed issue with builds list not updating when changing between [[Heroes|heroes]] in sandbox

* Added localization support for German, Polish, and Korean.

* Matchmaking window is now open 1 hour earlier on weekdays and weekends

* Players can only pause once per game
* A team can only pause at most 3 times in total amongst all the players
* The game application now flashes when the game is unpaused

* Various client and server performance improvements

* Added streamer setting to the options panel. Causes player names in the game UI panels to be hidden.

* Added a [[Melee]] Trainer bot in the 'gym' area of [[Hero|hero]] sandbox

* Added new "Game out of Date" status message at the top of the hud, so you can quickly know when there is an update rather than finding out at queue time
* Added a checkbox on the post game survey to make it not show again in the future
* [[Items]] now show their [[Spirit Power|spirit power]] impact when using alt
* Fixed profile match history not remembering the history page you were on
* Added Heal [[Amp]] and [[Debuff Resist]] to the [[Vitality]] [[Stats|stats]] in the [[Shop|shop]]
* [[Hero]] details on the dashboard now show [[Spirit|spirit]] information with alt, rather than requiring you to be in-game (this was added in a hotfix patch recently)
* The game will now prevent any quick [[Ability|ability]] upgrades for 2 seconds after [[Death|death]], to help prevent against accidental upgrades
* Can now reliably spectate a friend's game via the Friendlist
* Miscellaneous fixes to friends and party menus
* Added unavailable state to Zipline UI when zipline is on [[Damage|damage]] [[Cooldown|cooldown]]
* Spectating controls now use Left/Right mouse buttons to switch between players on a team and space to switch [[Teams|teams]]. This now works with free cursor mode, and you can still click on portraits/[[Minimap|minimap]] to spectate individual [[Heroes|heroes]].
* Added new [[Respawn|respawn]] countdown [[Music|music]] 5 seconds before [[Respawn|respawn]]
* Added alternate [[Shop|shop]] [[Music|music]] which plays in the [[Neutral|neutral]] [[Shop|shop]]
* Spectator count is now shown in-game
* Player names are now shown to spectators
* Added object motion blur
* Optimized performance of distance field ambient occlusion

* Fixed selling an [[Item|item]] not removing [[Charges|charges]] correctly. Now you can't buy {{ItemIcon|Extra Charge}} and immediately sell it and keep that readied charge.
* Fixed some tooltips not showing [[Golden Statue|golden statue]] bonuses if it wasn't the direct stat (i.e. you got [[Fire Rate|fire rate]] but bullets per second didn't show the [[Golden Statue|golden statue]])
* Fixed {{ItemIcon|Sharpshooter}} showing up in [[Stats|stats]] for [[Bullet Velocity|bullet velocity]] even though it doesn't increase your [[Bullet Velocity|bullet velocity]]
* Fixed a bug that caused [[Items|items]] in the build page to not be dimmed when the [[Shop|shop]] was out of range
* Fixed bug with {{HeroIcon|Lash}} not hearing his own {{AbilityIcon|Ground Strike}} impact explosion sound
* Added new effects for [[Withering Whip]]
* Updated {{ItemIcon|Ethereal Shift}} effects
* Fixed the disarmed spinner not showing up on your crosshair when you get disarmed
* Fixed parts of {{HeroIcon|Shiv}}'s coat being considered a [[Headshot|headshot]]
* Build list will now refresh if you change [[Heroes|heroes]] in the sandbox
* Fixed Zipline speed from [[Base Guardian]] killing and Zip booster [[Ability|ability]] not stacking properly
* Fixed [[Base Guardian]] boost not properly doing its ramp up and instead instantly being fast
* Added custom effects for Abram's {{AbilityIcon|Seismic Impact}} T3 buff
* Fixed a bug where someone could be holding down M1 and still be attacking when the magic carpet arrives
